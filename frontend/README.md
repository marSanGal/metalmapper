# MetalMapper Frontend

React frontend for MetalMapper built with Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

To build for production:
```bash
npm run build
```

## Features

- User authentication (login/register)
- Browse metal bars
- View bar details
- Check in at bars
- Rate bars
- Suggest new bars
- Admin panel for managing suggestions
